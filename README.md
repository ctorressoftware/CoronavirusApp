# CoronavirusApp

Aplicación Web ASP .NET MVC que permite visualizar las últimas noticias sobre el coronavirus de diariolibre.com

#### Se utilizaron las siguientes tecnologías:
- ASP .NET MVC
- HTML5, CSS3 y Bootstrap 
- Biblioteca Syndication
- PagedList Mvc
